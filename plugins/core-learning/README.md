# core-learning

Learn Java frameworks from the **inside out** by analyzing their internal core and building simplified reimplementations that extend outward.

## Skills

| Skill | Purpose |
|-------|---------|
| `core-blueprint` | Analyzes a framework repo and produces an inside-out learning outline (`core-outline.md`) |
| `core-builder` | Implements a single feature as working code + tests + tutorial chapter |

## Agents

| Agent | Role |
|-------|------|
| `core-code-explorer` | Traces internal hierarchies, integration points, and code patterns |
| `core-code-architect` | Designs simplified component architectures and integration strategies |
| `core-code-reviewer` | Validates correctness, test quality, and tutorial accuracy |

## Usage

```
# Step 1: Generate the learning outline
/core-blueprint /path/to/framework-repo

# Step 2: Implement features one by one
/core-builder "Bean Lifecycle"
```

## Prerequisites

- Java 17+
- Maven or Gradle
- A locally cloned Java framework/library repository
